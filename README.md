# 🔬 AI-Powered Research Agent

An intelligent, multi-agent AI research companion built using **IBM Bob**, **IBM watsonx.ai**, and **IBM Granite Models**. This application automates literature synthesis, analyzes multimodal inputs (PDFs, notes), predicts emerging research trends, and highlights citation gaps.

---

## 📌 Features

- **Academic Synthesis & Literature Reviews:** Aggregates research papers, preprints, and journals to generate structured literature reviews.
- **Multimodal Input Processing:** Accepts user queries, academic notes, charts, and PDF documents to extract key findings, methodologies, and limitations.
- **Predictive & Agentic Layer:** Automatically identifies research gaps, evaluates limitations, and suggests future research directions in a given domain.
- **Retrieval-Augmented Generation (RAG):** Grounds answers directly in uploaded scientific papers to eliminate hallucinations and ensure factual accuracy.

---

## 🛠️ Tech Stack

- **Platform / Orchestration:** IBM Bob / IBM watsonx.ai
- **LLM / Foundation Models:** IBM Granite Models (`granite-3-8b-instruct`)
- **Knowledge Base & Retrieval:** Vector Database (RAG pipeline for paper indexing)
- **Frontend / Dashboard:** Interactive HTML / Web Interface

## 🚀 Getting Started

1. Open `research_agent.html` in any web browser to access the research assistant.
2. Upload academic PDFs or enter research topics/queries into the agent interface.
3. Review generated literature summaries, key methodologies, limitations, and suggested future research directions.
